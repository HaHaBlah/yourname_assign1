Scroll to Top Button

A button that when clicked, scrolls all the way back to the top of the page

HTML

<div class="scroll-top">
   <a class="scroll-top__link" href="#">& #11165;</a>
</div>

CSS

.scroll-top {
   position: sticky;
   bottom: 2rem;
   height: 0;
   width: 100%;
   z-index: 10;
   margin-top: -50px;
   text-align: right;
}

.scroll-top__link {
   position: fixed;
   bottom: 2rem;
   right: 2rem;
   pointer-events: auto;
   display: inline-block;
   text-decoration: none;
   border-radius: .5rem;
   background-color: var(--accent-color-1);
   padding: 0 .5rem;
   color: white;
   font-size: 2em;
   opacity: .5;
   z-index: 10;
   transition: opacity 0.1s ease-in-out;
}

.scroll-top__link:hover {
   opacity: 1;
}

Flip card effects

When an object was hovered, it will flip

HTML

<div class="location-card-inner">
    <div class="location-card-front">
        <img src="#" alt="Front Image">
    </div>
    <div class="location-card-back">
        <a href="https://maps.app.goo.gl/8ap6MykG7yn7rF7A6">
            <img src="images/location/Location_1.png" alt="Location 1 Back Image">
        </a>
    </div>
</div>

CSS

.location-card-inner {
    position: relative;
    width: 100%;
    height: 100%;
    transition: transform 0.6s;
    transform-style: preserve-3d;
    transform: rotateY(180deg);
}

.location-card-front, .location-card-back {
    position: absolute;
    width: 100%;
    height: 100%;
    -webkit-backface-visibility: hidden;
    backface-visibility: hidden;
}

.location-card-front {
    background-color: var(--accent-color-1);
    color: white;
}

.location-card-back {
    transform: rotateY(180deg);
}

.location-card img {
    width: 100%;
    height: 100%;
    object-fit: cover;
    display: block;
    border-radius: 50%;
    overflow: hidden;
}

Color Change when Hover effect

When an object was hovered, it will change color

HTML

<div class="sci">
    <a href="#" target="_blank">
        <img src="#" alt="Icon">
    </a>
</div>

CSS

.sci {
    margin-left: 20px;
}

.sci a {
    display: inline-flex;
    padding: 8px;
    border: 2px solid var(--accent-color-1);
    border-radius: 50%;
    font-size: 20px;
    height: 40px;
    width: 40px;
    color: var(--accent-color-1);
    margin: 0 8px;
    transition: .25s;
    box-sizing: border-box;
}

.sci a img {
    filter: invert(22%) sepia(18%) saturate(1418%) hue-rotate(343deg) brightness(94%) contrast(86%);
}

.sci a:hover {
    background: var(--accent-color-1);
    color: #A99175;
    box-shadow: 0 0 10px var(--accent-color-1);
}

.sci a:hover img {
    filter: invert(64%) sepia(29%) saturate(331%) hue-rotate(352deg) brightness(87%) contrast(90%);
}

Animated conic gradient border

Moving Border effect around the profile photos

HTML

<div class="home-img">
    <div class="img-box">
        <div class="img-item">
            <img src="#" alt="#">
        </div>
    </div>
</div>

CSS

.home-img {
    flex: 0 0 auto;
    display: flex;
    justify-content: flex-end;
    align-items: center;
    width: auto;
}

.home-img .img-box {
    position: relative;
    width: 32vw;
    height: 32vw;
    border-radius: 50%;
    padding: 5px;
    display: flex;
    justify-content: center;
    align-items: center;
    overflow: hidden;
}

.home-img .img-box::before,
.home-img .img-box::after {
    content: '';
    position: absolute;
    width: 500px;
    height: 500px;
    background: conic-gradient(transparent, transparent, transparent, var(--accent-color-1));
    transform: rotate(0deg);
    animation: rotate-border 16s linear infinite;
}

.home-img .img-box::after {
    animation-delay: 8s;
}

@keyframes rotate-border {
    100% {
        transform: rotate(360deg);
    }
}

.home-img .img-box .img-item {
    position: relative;
    width: 100%;
    height: 100%;
    background: var(--accent-color-1);
    border-radius: 50%;
    border: .1px solid var(--accent-color-1);
    display: flex;
    justify-content: center;
    z-index: 1;
    overflow: hidden;
}

.home-img .img-box .img-item img {
    position: absolute;
    top: 1px;
    display: block;
    width: 99.7%;
    object-fit: cover;
}
